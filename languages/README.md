> [!IMPORTANT]
> Before adding a new translation, choose a **unique language key** for the language file.
>
> For example, if you're translating Hindi and want to use `hi.js`, first check whether a file with that name already exists in this directory.
>
> If the key is already taken, simply choose another unique one, such as `hindi.js`, `hin.js`, `india.js` or any other unused name. The actual filename is not important, **uniqueness and consistency are**.
>
> Once you've chosen a language key, use it **everywhere** it is referenced throughout the project. The filename and all related  references must match exactly.
